What rates do we have at Arkadiark?

Here you can see all the features of the server. The map is Ragnarok. There is no ORP.

  • Harvest x5
  • Experience x10
  • Taming x15
  • Breeding x20
  • Max dino level 180
  • Max player level 100

What rules do we have at Arkadiark?

There are no rules except that you are not allowed to use any kind of hacks or use bugs, and also all kinds of discriminatory comments are forbidden. If you have any doubts, please contact to an admin.

Which mods do we have at Arkadiark?

Arkadiark has the best and most-fashioned mods:

  • S+ Structures
  • HG Stacking
  • Better Reusables
  • Classic Flyers
  • Universal Server UI (Zelur)

From where are we hosting?

Our server is hosted in Spain, in one of the best existing DPC, providing a seamless experience to residents in Spain and the rest of Europe.
